# Node.js Book Management API

This is a RESTful API built using Node.js for managing books. It allows users to perform CRUD operations on books, and the data is stored in a MongoDB database.

## Features

- **Create a new book:** Add a new book with title, author, and summary.
- **View all books:** Retrieve a list of all books.
- **View book details by ID:** Get detailed information about a specific book using its ID.
- **Update a book:** Modify a book's details.
- **Delete a book:** Remove a book from the database.

## Technologies Used

- Node.js
- Express.js
- MongoDB (using MongoDB Atlas/cloud solution or local instance)
